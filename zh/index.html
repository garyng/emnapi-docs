emnapi

适用于 Emscripten 的 Node-API 子集实现

开始使用

可移植

几乎不需要更改代码即可在浏览器上运行你的 Node-API 原生模块。

轻量级

微型运行时和按需链接 JavaScript 实现。

多线程

通过 Web Worker 支持多线程异步操作（async work，线程安全函数）。
